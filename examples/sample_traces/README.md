# Sample Traces

This folder contains curated execution traces that demonstrate the controlled agentic orchestration layer.

The traces show:
- profile-specific plans;
- selected tools;
- coverage scores;
- coverage gaps;
- retry decisions;
- source mix;
- generated output paths.

These are execution traces, not hidden chain-of-thought traces.

Included examples:
- `financial_services_trace.json`
- `financial_services_gap_trace.json`
- `ai_technology_strategy_trace.json`
